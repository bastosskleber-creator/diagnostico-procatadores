Diagnóstico da Prefeitura - T0
Município: Município não informado   |   Padrão de resposta: SIM, PARCIALMENTE, NÃO ou NÃO SE APLICA.
Painel Executivo da Prefeitura - T0
Resumo Prefeitura: leitura executiva da maturidade municipal, indicadores territoriais, gargalos, potencialidades e análise técnica por aspecto.
Plano de Ação da Prefeitura - T0
Plano de Ação Prefeitura: gerado a partir das respostas NÃO e PARCIALMENTE do diagnóstico municipal. O consultor pode editar ações, responsáveis, prioridade, prazo, status e observações antes de salvar.
Cadastro de Soluções Prefeitura
Soluções Prefeitura: cadastro das soluções técnicas, oficinas, consultorias, capacitações e ações de apoio voltadas à gestão municipal de resíduos sólidos. Este módulo é independente do cadastro de soluções da Cooperativa.

Backup e Restauração
Backup por entidade: escolha exatamente o que deseja salvar. Os botões abaixo separam Cooperativa, Prefeitura e Território para reduzir dúvidas em campo.

Backup Cooperativa

Use para salvar somente os dados da Cooperativa.

Backup Prefeitura

Use para salvar somente os dados da Prefeitura.

Backup Territorial

Use para salvar todo o município em um único arquivo: Cooperativa, Prefeitura, T0, T1, planos e soluções.

Base de teste: use este botão para preencher automaticamente dados fictícios de Cooperativa e Prefeitura, incluindo T0 e T1, planos de ação e soluções. Essa opção é indicada apenas para demonstração e validação pelos consultores.
Importação: use este botão para restaurar arquivos de backup individual, por entidade ou territorial.
